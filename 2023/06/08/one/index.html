这是一个用Hexo搭建的个人博客并上传至GitHub的教程。

Hexo简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。这是官网：Hexo 。

接下来我们继续在CMD中输入：

hexo g: 初始化你的Hexo

hexo s -p 5555 ： 在浏览器上输入http://localhost:5555 即可看到Hexo初始博客界面，其中5555是设置的端口号，可以填写其他的，比如：6666，4444等

ctrl + c即可关闭服务

创建Github仓库

首先，我们需要在GitHub官网 注册一个账户。然后，登录你的账户并点击页面右上角的头像，选择your repository，点击绿色字体的New，会看到以下界面：

Repository name ： 输入你的仓库名（你的用户名.github.io），其他的不用管。

然后点击create repository完成创建

创建成功后点击setting：

并点击左列中的Pages

在Branch中选择main和/root，并保存

这样我们在浏览器中输入 你的仓库名即可访问（注意，你的仓库名必须是： .github.io形式的）

接下来我们需要通过SSH进行仓库代码的快速部署，如果你的电脑已经安装过git并绑定过GitHub仓库，则这一步可以跳过。

在电脑上找到你git的安装目录，或者点击windows自带的搜索框找到安装在电脑上的Git Bash

在git bash中输入：

youremail是你创建Github的邮箱地址

然后回车输入密码（一直回车）

在此电脑中，进入C盘根目录下，找到用户文件夹，并打开你电脑当前用户文件夹，找到.ssh文件夹并打开id_rsa.pub这个文件（用记事本打开这个文件），复制里面的内容
